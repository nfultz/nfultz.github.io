Grep notes

  • grep -F :: fixed string
  • grep -v :: invert pattern
  • grep -q :: disable output and early exit.

This note that grep -vq succeeds when the file has a non-matching line. I almost always instead want !grep -q which succeeds when there is no matching line. De Morgans Law is a hell of a drug.
