SVG Icons

Instead of using fontawesome, can get SVG icons from iconmonstr.com. Each SVG is a one-line SVG document. They can be combined into a single one if you set an "id" attribute on each path tag. Then you can refer to each via a <USE> tag - see the source of front.html for an example.

See also seriously-dont-use-icon-fonts

Frames => IFRAMES

Frames are the original implementation of nested web pages. IFrames are the newer implementation with slightly better behavior with the back button.

See this commit for an example. Basically, the old styling provided by FRAMESET has to be emulated using CSS.
