Bridging audio

$pacmd load-module module-virtual-source source_name=VirtualMic
$pacmd load-module module-null-sink sink_name=MySink

In OBS - audio advanced properties - turn on monitor for sources Then in pavucontrol, playback tab - change all obs monitors to null output pavucontrol recording tab - change virtualmic to monitor of null output

zoom - audio - use virtual mic

Note that this is laggy as hell.

Cleanup:

$pacmd list-modules
$pacmd unload-module 35
